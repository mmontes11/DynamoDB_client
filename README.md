DynamoDB_client
===============

A sample NodeJS + Express + Jade client app for Amazon DynamoDB

## Installation

1. Replace 'XXXX' for your Amazon AccessKeyId and SecretAccessKey in ddb/ddb.js .

2. Install NodeJS and npm and execute: 
  npm install
  npm start
